Where can I find my Channel ID?

To find a Channel ID, you'll first need to enable Developer Mode in your Discord settings. Once enabled, you can copy the ID directly from the channel.

How to Enable Developer Mode

In order to be able to copy an ID number, you need to have Developer Mode enabled. Here's how.

Finding Channel ID on Desktop

1
Enable Developer Mode

First, make sure Developer Mode is enabled following the steps above.

2
Copy Channel ID

Once Developer Mode is enabled:

  • Navigate to the channel whose ID you want to find
  • Right-click on the channel name in the channel list
  • Select Copy ID from the context menu
Right-click on a channel and select "Copy ID"
Note

The Channel ID is now copied to your clipboard and can be pasted anywhere you need it.

Finding Channel ID on Mobile

1
Enable Developer Mode

First, make sure Developer Mode is enabled following the steps above.

2
Copy Channel ID

Once Developer Mode is enabled:

  • Navigate to the channel whose ID you want to find
  • Tap and hold on the channel name in the channel list
  • Select Copy ID from the menu that appears
Tap and hold on a channel and select "Copy ID"
Note

The Channel ID is now copied to your clipboard and can be pasted anywhere you need it.

What are Channel IDs used for?

Channel IDs are unique identifiers for Discord channels. They're commonly used for bot development, API integrations, creating channel mentions in messages, and troubleshooting.
