Where can I find my User ID?

To find a User ID, you'll first need to enable Developer Mode in your Discord settings. Once enabled, you can copy the ID directly from the user.

Enable Developer Mode First

Before you can copy User IDs, you need to enable Developer Mode in your Discord settings. If you haven't done this yet, please follow our guide on enabling Developer Mode first.

Note

You can copy user IDs from a server, group chat, or direct message with a user.

Finding User ID on Desktop

1
Navigate to where you want to copy the user ID from.

This could be a server member list, group chat, or direct message.

2
Next, right-click on the user and tap on Copy User ID.

Right-click on the user's name in the member list or chat.

3
After copying the user ID you can then paste it where needed.

The User ID is now copied to your clipboard and can be pasted anywhere you need it.

Desktop: Right-click a user to find Copy User ID

Finding User ID on Mobile

1
Navigate to where you want to copy the user ID from.

This could be a server member list, group chat, or direct message.

2
Tap on the user and then tap on the ellipsis in the top-right corner.

First tap on the user's name to open their profile, then look for the ellipsis menu.

3
Next, tap on Copy User ID.

Select "Copy User ID" from the menu that appears.

4
After copying the user ID you can then paste it where needed.

The User ID is now copied to your clipboard and can be pasted anywhere you need it.

Mobile: Tap a user to find Copy User ID
What are User IDs used for?

User IDs are unique identifiers for Discord users. They're commonly used for bot development, API integrations, reporting users to Discord support, and various administrative functions.
