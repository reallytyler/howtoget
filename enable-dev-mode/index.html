How to Enable Developer Mode on Discord

Developer Mode is required to copy IDs (User, Channel, Server, Message) on Discord. This guide will show you how to enable it on both desktop and mobile.

What is Developer Mode?

About Developer Mode

Developer Mode adds additional options to Discord's right-click menus, allowing you to copy IDs for users, channels, servers, and messages. These IDs are essential for bot development, API integrations, and various administrative tasks.

Enable Developer Mode on Desktop

1
Tap on the in the bottom left corner of the desktop app to open up your User Settings.
2
Then head to Advanced in the left sidebar.
3
Now, tap on the main toggle next to Developer Mode to enable.

A ✓ means it's enabled, while an ✗ means it's disabled.

Desktop: Open User Settings > Advanced and toggle on Developer Mode

Enable Developer Mode on Mobile

1
Tap on your Avatar in the bottom-right corner.
2
Next, tap on the in the top-right corner.
3
Now, scroll down until you see Advanced under App Settings.
4
Finally, tap on the toggle next to Developer Mode.

A ✓ means it's enabled, while an ✗ means it's disabled.

Mobile: Open Settings > Advanced and toggle on Developer Mode

What's Next?

Ready to Find IDs

Now that you've enabled Developer Mode, you can:

  • Find User IDs - Copy IDs for specific users
  • Find Channel IDs - Copy IDs for text and voice channels
  • Find Server IDs - Copy IDs for entire servers
  • Find Message IDs - Copy IDs for specific messages
Important Notes

Developer Mode needs to be enabled separately on each device where you want to copy IDs. Enabling it on desktop doesn't automatically enable it on mobile, and vice versa.
